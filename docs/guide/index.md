# Deployment Guide

This guide walks you through the complete installation of an on-premise LLM server, from hardware procurement to production deployment.

## Prerequisites

- Basic Linux administration experience
- Access to enterprise hardware procurement
- RHEL subscription or evaluation license
- Network access for software downloads

## Steps Overview

| Step | Description | Time Estimate |
|------|-------------|---------------|
| [Step 1: Hardware](step1-hardware.md) | Hardware procurement and assembly | 1-2 weeks |
| [Step 2: RHEL](step2-rhel.md) | Red Hat Enterprise Linux installation | 2-4 hours |
| [Step 3: NVIDIA](step3-nvidia.md) | NVIDIA driver installation | 1-2 hours |
| [Step 4: App Stack](step4-apps.md) | Software stack deployment | 2-3 hours |
| [Step 5: WebUI & vLLM](step5-webui.md) | Open WebUI and vLLM deployment | 1-2 hours |
| [Step 6: AD Integration](step6-ad.md) | Active Directory integration | 2-4 hours |

**Total Estimated Time**: 2-3 days (excluding hardware procurement)

## Quick Start

1. Start with [Step 1: Hardware](step1-hardware.md) to understand hardware requirements
2. Follow each step sequentially
3. Refer to the [Appendix](../appendix/index.md) for troubleshooting and additional notes
